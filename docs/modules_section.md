# Modules - section
> The section module provides methods to display section information.

- [list](#list)
- [allobjects](#allobjects)

## list
List all sections.
```sh
$ eep section list
```

## allobjects
Lists all content objects in the section.
- supports ```--limit=N``` and/or ```--offset=M```

```sh
$ eep section allobjects <section id>
```
